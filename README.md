# Smart Water Level Controller

## Project Description
The Smart Water Level Controller is a web-based application that helps monitor and control water levels in a tank. It provides real-time water level information and helps prevent water overflow and water shortage.

## Features
- Water level monitoring
- Tank status display
- User-friendly interface
- Responsive design
- Real-time updates

## Technologies Used
- HTML
- CSS
- JavaScript
